0 工作原理

1

SDK 初始化时无需 API Key，直接创建客户端实例

2

调用 autoRegister() 自动注册，获取 API Key

3

重要：将 API Key 保存到你的业务系统（数据库/配置文件）

4

下次使用时，用保存的 API Key 初始化客户端，记忆数据会自动关联

记忆数据跟着 API Key 走。同一个 API Key 在任何设备上都能访问相同的记忆。

1 Node.js SDK

安装

npm install @t0ken.ai/memoryx-sdk

首次使用 - 自动注册获取 API Key

import { APIClient } from '@t0ken.ai/memoryx-sdk';

// 首次使用：无需 API Key
const client = new APIClient();

// 自动注册，获取 API Key
const account = await client.autoRegister('my-app', 'My Application');

console.log('API Key:', account.api_key);
console.log('Project ID:', account.project_id);

// ⚠️ 重要：保存 API Key 到你的数据库或配置文件！

后续使用 - 用保存的 API Key

import { APIClient } from '@t0ken.ai/memoryx-sdk';

// 从你的存储中读取保存的 API Key
const savedApiKey = await db.getApiKey(userId);

// 使用保存的 API Key 初始化
const client = new APIClient({ apiKey: savedApiKey });

// 添加记忆
await client.sendMemories([
  { content: '用户偏好深色主题', metadata: { category: 'semantic' } }
]);

// 搜索记忆
const results = await client.search('用户偏好');

主要方法

autoRegister() 自动注册获取 API Key
sendMemories() 发送记忆（单条/批量）
search() 语义搜索记忆
list() 获取记忆列表
delete() 删除记忆

2 Python SDK

安装

pip install t0ken-memoryx

首次使用 - 自动注册获取 API Key

from memoryx import APIClient

# 首次使用：无需 API Key
client = APIClient()

# 自动注册，获取 API Key
account = client.auto_register("my-app", "My Application")

print(f"API Key: {account['api_key']}")
print(f"Project ID: {account['project_id']}")

# ⚠️ 重要：保存 API Key 到你的数据库或配置文件！

后续使用 - 用保存的 API Key

from memoryx import APIClient

# 从你的存储中读取保存的 API Key
saved_api_key = db.get_api_key(user_id)

# 使用保存的 API Key 初始化
client = APIClient(api_key=saved_api_key)

# 添加记忆
client.send_memories([
    {"content": "用户偏好深色主题", "metadata": {"category": "semantic"}}
])

# 搜索记忆
results = client.search("用户偏好")

主要方法

auto_register() 自动注册获取 API Key
send_memories() 发送记忆（单条/批量）
search() 语义搜索记忆
list() 获取记忆列表
delete() 删除记忆

3 记忆分类

episodic

情景记忆 - 特定事件和经历

semantic

语义记忆 - 事实和概念

procedural

程序记忆 - 技能和操作方法

emotional

情感记忆 - 用户偏好和情感

reflective

反思记忆 - 总结和反思内容
